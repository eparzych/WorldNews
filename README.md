# gnNews
The application was created as part of the challenge and is used to display current news from various countries. Live demo [_here_](https://gn-news.herokuapp.com)

## Technologies Used
- React
- Redux Tooltkit
- React Router
- React Testing Library
- Typescript
- react-modal
- react-world-flags
- react-icons
- i18next
- GRID
- SCSS
- REST API
- Crate-React-App

## Setup
- npm install
- npm start
